# Interpolation-pygame

A simple Pygame application demonstrating smooth animations using interpolation and easing functions.

## Overview

This project provides a reusable `Interpolated` class that animates values over time using various easing functions.

## Features

- **Interpolated Class**: Smoothly interpolates between start and end values over a specified duration
- **Easing Functions**: Collection of easing functions from [easings.net](https://easings.net).
- **Pygame Integration**: Simple visualization of interpolation in action 

## Project Structure

- `easings.py` - Easing functions for controlling animation curves
- `interpolate.py` - Core `Interpolated` class for value interpolation
- `main.py` - Example application showing a moving circle

## Requirements

- Python 3.x
- Pygame
